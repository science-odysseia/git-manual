# LATEX 수식 

수식 기본 표현 : 양 끝에 **띄어쓰기 없이** $를 붙인다.

    $a + b = c$

$a + b = c$

아래와 같이 하면 중앙에 식이 생긴다.

    $$
    a + b = c
    $$

$$
a + b = c
$$

## 기본연산자

|기본연산자|수식표현|
|:---:|:---:|
|덧셈| `a + b` |
|뺄셈| `a - b` |
|곱셈| `a \cdot b` : '.'기호, `a \times b` : 'X' 기호|
|나눗셈| `\frac{분자}{분모}` |
|지수| `a^b` |
|아래첨자| `a_n` |
|루트| `sqrt{x}`, `sqrt[n]{x}` = `x ^ (1/n)`  |
