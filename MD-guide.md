# 1. 제목, 부제목, 소제목

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

작성 방법

    # 제목1
    ## 제목2
    ### 제목3
    #### 제목4

\#을 제목기호가 아닌 문자 그대로 사용하려면 \\#으로 쓰면 된다.

------------------------------------------------------------------------

# 2. 문단과 줄바꿈

Markdown은 그냥 엔터만 치면 줄바꿈이 안되는 경우가 있다.

방법 1 (공백 두 칸)

    첫번째 줄(공백)(공백)  
    두번째 줄

결과

첫번째 줄  
두번째 줄

방법 2

    첫번째 줄<br>
    두번째 줄

결과

첫번째 줄<br>
두번째 줄

방법 3 (일부 환경에서 가능한 방법)

일부 Markdown 렌더러에서는 백슬래시 \로 줄바꿈이 가능하다.

    첫번째 줄\
    두번째 줄

결과 (지원하는 환경에서)

첫번째 줄\
두번째 줄

# 4. 목록

순서 없는 목록

    -(공백필수)item1
    -(공백필수)item2
      -(공백필수)sub item
        -(공백필수)sub item2

결과

-   item1
-   item2
    -   sub item
        -   sub item2

순서 있는 목록

    1.(공백필수)first
    2.(공백필수)second
    3.(공백필수)third

결과

1. first
2. second
3. third

# 부가적인것들(알 필요 없는 것들)

들여쓰기

    &nbsp;한개당 한칸 들여쓰기

예시(15칸)

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;이 문장은 들여쓰기된 문장입니다.


내가 어떤 코드를 md상에서 쓰고 싶은데

md상에서 코드를 코드가 아닌 글자 그대로 쓰고 싶다면

md상에서 들여쓰기 한번 해주면 된다.

예를 들어 

&nbsp;&nbsp;&nbsp;&nbsp;\*\*굵게\*\*

이런 식으로 하면 된다.

특수문자 \#, \* 같은 경우도 \\#, \\* 이렇게 처리하면 된다.

혹은 ``로 둘러싸 버리는것도 방법이다.

md는 코드 제외하고 모든 줄 사이 빈줄 1개씩 넣는걸 무조건 추천한다.

안그러면 엔터 쳤음에도 불구하고 줄이 안 바뀌거나 혹은 꼬일 수 있다.
