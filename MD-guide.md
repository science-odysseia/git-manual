# 1. 제목 (Heading)

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

작성 방법

    # 제목1
    ## 제목2
    ### 제목3
    #### 제목4

\#을 제목기호가 아닌 문자 그대로 사용하려면 \\#으로 쓰면 된다.

------------------------------------------------------------------------

# 2. 문단과 줄바꿈

Markdown은 그냥 엔터만 치면 줄바꿈이 안되는 경우가 있다.

방법 1 (공백 두 칸)

    첫번째 줄(공백)(공백)  
    두번째 줄

결과

첫번째 줄  
두번째 줄

방법 2

    첫번째 줄<br>
    두번째 줄

결과

첫번째 줄<br>
두번째 줄

방법 3 (일부 환경에서 가능한 방법)

일부 Markdown 렌더러에서는 백슬래시 \로 줄바꿈이 가능하다.

    첫번째 줄\
    두번째 줄

결과 (지원하는 환경에서)

첫번째 줄\
두번째 줄

# 4. 목록

순서 없는 목록

    -(공백필수)item1
    -(공백필수)item2
      -(공백필수)sub item
        -(공백필수)sub item2

결과

-   item1
-   item2
    -   sub item
        -   sub item2

순서 있는 목록

    1.(공백필수)first
    2.(공백필수)second
    3.(공백필수)third

결과

1. first
2. second
3. third

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;이 문장은 들여쓰기된 문장입니다.

# 5. 코드

한 줄 코드

    `print("hello")`

여러 줄 코드

    ``` python
    print("hello")
    print("world")
    ```
<br>

    ``` bash
    sudo apt update
    sudo apt upgrade
    ```

결과

한 줄 코드(한 줄 코드의 경우 붙여넣기 버튼이 없음)

`print("hello")`

여러 줄 코드

``` python
print("hello")
print("world")
```

``` bash
sudo apt update
sudo apt upgrade
```

언어를 지정하면 **GitHub에서 syntax highlight**가 된다.

# 7. 이미지

    ![설명](image.png)

레포지토리 폴더를 기준으로 경로 작성

그래서 이미지도 같이 업로드해야됨

이미지이름.확장자 => 이미지이름, 확장자 모두 완벽하게 일치해야함.

예시

```
![logo](cat.jpeg)
```
![logo](cat.jpeg)

logo의 경우 대체텍스트로, 이미지가 뜨지 않을 경우 이미지 대신 뜨는 텍스트.
